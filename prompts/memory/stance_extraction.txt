请分析以下我的回复，提取其中值得进入“主播自我记忆”的内容。

观众说：{input}
我回复了：{response}

目标不是只判断“有没有立场”，而是提取两类结构化内容：

1. self_said
- 我明确说过的主观看法、偏好、立场、自我声明
- 例如“我觉得XX更好”“我一般都选XX”“我是XX派的”

2. commitments
- 我明确说过的承诺、打算、后续动作、未完的话头
- 例如“我以后会XX”“下次我再讲XX”“我之后补你这个”

以下不算：
- 纯客观知识介绍
- 普通问候和感叹
- 简单附和观众
- 与“我”无关的空泛描述

请严格输出 JSON，不要输出其他任何内容。

没有可保留内容时：
{{"has_memory": false}}

有内容时：
{{
  "has_memory": true,
  "self_said": [
    {{
      "topic": "话题关键词",
      "statement": "用第一人称概括我说过的话"
    }}
  ],
  "commitments": [
    {{
      "topic": "话题关键词",
      "text": "我承诺/打算/延续的话",
      "status": "open"
    }}
  ]
}}

兼容要求：
- `statement` 尽量保留“我”的视角
- 只提取真的值得长期一致化的内容
- 不确定时宁可少提，不要臆造
